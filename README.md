# contact_form
Desafio do FrontEnd Mentor. <br>

Página responsiva de contato de formulário feito com HTML, CSS, JavaScript.

Durante o desenvolvimento deste formulário, enfrentei alguns desafios importantes:

• Tive dificuldades no alinhamento dos campos e espaçamento entre os inputs utilizando Flexbox.

• Aprendi a exibir uma mensagem de sucesso após o envio do formulário posicionando com o position e transform. 

• Aprendi também a implementar a validação de formulário com o JavaScript incluindo exibição de mensagens de erro especificas para cada campo. 

Para superar esses desafios utlizei pesquisas e ferramentas de apoio como o Gemini Code Assist integrado ao VS Code.
